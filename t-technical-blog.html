What is Boxifying Design

Boxifying design is dividing a mockup into rectangle boxes so that it aligns with the Box Model

What is the Box Model

The Box Model is a concept in which every element on a page is a rectangular box and may have width, height, padding borders, and margins.

Because there are multiple widths and heights based on the width and height of each of the content, the border and the padding elements, the boz sizing property simplifies things by defining which width/height is fixed and adjusting other widths and heights accordingly.

What is responsive site design, and why is it important?

Responsive site design is adding or referencing some css that adjusts your webite width and other features dependant on the size of the screen (in pixels) that is viewing your page.

What is mobile first design, and why is it important?

A higher percentage of web content is getting viewed primarily or only on mobile phones and similar smaller devices such as tablets. Mobile first design is gesigning your site with mobile in mind and working outwards, in terms of screen size, to the traditional desktop sized resolution.

What are frameworks, and what are the pros and cons.

Frameworks for webpages are similar to frameworks for building a house. Each time a builder builds a house they use commonly accepted and standard methods for constructing the frame of the house. Similarly web developers don't need to start from scratch or reinvent the wheel everytime they start a new webpage, they can use pre-constructed packages of html, css and js as a starting point for the new webpage.

How does JavaScript compare to HTML and CSS?

HTML puts the content on the webpage (text, images, lists, tables et), CSS makes it look good (headers, backgrounds, borders etc) and Javascript makes it interactive (logins, popups, database queries etc)

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control flow is the order in which individual statements, instructions or function calls of an imperative program are executed or evaluated.

In basketball, players follow moves similarly using a control flow, Billy gets open, Jonny passes to Billy, Jonny cuts, Billy looks for a pass to Jonny.

A loop is a sequence of instruction's that is continually repeated until a certain condition is reached.

In the basketball example above, if Billy can't get a pass to Jonny, Bobby gets open, Billy passes to Bobby, Billy cuts and Bobby looks for a pass to Billy.

The loop continues until it is broken by a certain condition. For example, a pass to the cutting player, a steal by the opposition, time is up.

Explain the difference between accessing data from arrays and objects.

Objects store data via a reference title (parameter), ie name, height, sex. Arrays store data via a reference position identified by these brackets [], starting with position [0]

To access data from an object you ask for the value associated with a specific parameter, ie, what is Billy's height.

To access data from an array, you need to know what position the data is in the array. ie, for Billy's height we would ask for the value stored in position [1] of the array (as 'name' is in position [0])
