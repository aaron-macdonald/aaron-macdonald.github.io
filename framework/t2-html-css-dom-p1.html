HTML, CSS and the DOM

Webpages consist of HTML (the meat and potatoes), CSS (the plating) and the DOM (what the diner receives)

What is Boxifying Design

Boxifying design is dividing a mockup into rectangle boxes so that it aligns with the Box Model

What is the Box Model

The Box Model is a concept in which every element on a page is a rectangular box and may have width, height, padding borders, and margins.

Because there are multiple widths and heights based on the width and height of each of the content, the border and the padding elements, the boz sizing property simplifies things by defining which width/height is fixed and adjusting other widths and heights accordingly.
