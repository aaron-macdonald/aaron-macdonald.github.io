What is grid based design

Grid based design is organising a web page so that it is appealing to the user via a top-down, left to right, grid based flow.

Whats all the hype about responsive webpages?

Responsive webpages are required as there are many different devices viewing webpages.

What is semantic structure?

Semantic structure is using tags that make sense to the user and the browser. They help to ensure the code is formatted in a way that is easily understandable to another user.
