Icons

Icons

Materialize has included 932 Material Design Icons courtesy of Google. You can download them directly from the Material Design specs.

Usage

To be able to use these icons, you must include this line in the < head> portion of your HTML code.

< div>
< link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">>
< /div>

To use these icons, use the material-icons class on an element and provide the ligature as the text content.

< div class="col s6 m3">
< i class="material-icons dp48">apps< /i>
< span>apps< /span>
< /div>
< div class="col s6 m3">
< i class="material-icons dp48">bookmark< /i>
< span>bookmark< /span>
< /div>

Sizes

Sizes

To control the size of the icon, change the font-size: 30px property of your icon. Optionally you can use preset classes as shown below.

  • Sizes:
  • tiny: 1rem
  • small: 2rem
  • medium: 4rem
  • large: 6rem
